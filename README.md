# iFood (React)

Dashboard para gestão do estabelecimento e controle de pedidos estilo iFood.

## Ferramentas

- React
- React Router DOM
- Tailwind
- shadcn/ui
- React Query
- Radix

## Aprendizados importantes

- Autenticação JWT
- Caching e revalidação
- Conexão com back-end (API)
- Testes unitários
